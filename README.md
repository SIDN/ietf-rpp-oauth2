# OAuth 2.0 for RESTful Provisioning Protocol (RPP)

This repository contains the IETF draft for using OAuth 2.0 to secure RESTful Provisioning Protocol (RPP) API requests.

The [draft](https://github.com/SIDN/ietf-rpp-oauth2/blob/main/src/draft-wullink-rpp-oauth2.md) is authored using [mmark](https://mmark.miek.nl/) Markdown.

Contributions in the form of a Pull Request are welcome.

For generated output see:   
[Plaintext](https://sidn.github.io/ietf-rpp-oauth2/draft-wullink-rpp-oauth2.txt)  
[HTML](https://sidn.github.io/ietf-rpp-oauth2/draft-wullink-rpp-oauth2.html)  
[PDF](https://sidn.github.io/ietf-rpp-oauth2/draft-wullink-rpp-oauth2.pdf)  
[XML](https://sidn.github.io/ietf-rpp-oauth2/draft-wullink-rpp-oauth2.xml)

## Command Line Usage

Requirements:

- [mmark](https://mmark.miek.nl/)
- [xml2rfc](https://github.com/ietf-tools/xml2rfc#installation)

Formatted versions can be built using:

```sh
cd src
make all
```
